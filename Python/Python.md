# Python

## Python의 기초
Python 파일명은 끝에 `.py`가 되어야 한다.  
Python을 실행할때는 commandLine에 `python` 이라고 입력한다.  
Python을 종료할때는 `exit()` 이라고 입력한다.
  
파일에 한글내용을 쓰려면 스크립트 가장 위에 이렇게 적자.
```Python 
#-*-coding: utf-8-*-
```

## Python 첫 번째 프로그램
예제파일 `ex01.py`  
  
프린트하기
``` Python
print "text"

print 't"e"xt4'
```

# 이렇게하면 한줄주석
