# 27. 논리 외우기

## 진리 용어
파이썬에서는 다음 용어를 써서 무엇이 'True'나 'False'인지 결정 한다.  
컴퓨터에서의 논리란 이 기호화 변수의 결합이 프로그램의 어떤 지점에서 True인지 보는 것이다.  

- and
- or
- not 
- != (같지 않다)  
- == (같다)  
- >= (크거나 같다)  
- >= (작거나 같다)  
- True  
- False  

## 진리표

**NOT**
```
not False -> True  
not True -> False  
```
