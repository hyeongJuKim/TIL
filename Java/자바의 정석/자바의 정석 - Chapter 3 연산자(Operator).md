## Chapter 3 연산자(Operator)
### 1.연산자(operator)

| 종류      | 연산방향 | 연산자                                      | 우선순위 |
| ------- | ---- | ---------------------------------------- | ---- |
| 단항 연산자  | <-   | `++` `—` `+` `-` `~` `!` `타입`            | 높음   |
| 산술연산자   | ->   | `*` `/` `%`    `+` `-`   `<<` `>>` `>>>` | 높음   |
| 비교연산자   | ->   | `<` `>` `<=` `>=` `instanceof`    `==` `!=` | 높음   |
| 논리 ]연산자 | ->   | `&` `^` `|` `&&` `||`                    | 높음   |
| 삼항연산자   | ->   | `?:`                                     | 높음   |
| 대입연산자   | <-   | `=` `*=` `/=` …..                        | 낮음   |

1. 산술 -> 비교 -> 논리 -> 대입
2. 단항 -> 이항 -> 삼항
3. 단항 연산자와 대입연산자를 제외한 모든 연산의 진행방향은 왼쪽에서 오른쪽이다.
